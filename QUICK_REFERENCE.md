## 🎯 快速操作步骤

### 群发给多个群

```
1. 选菜单 → 1
2. 输入消息内容
3. 输入群ID (逗号分隔): group1@g.us,group2@g.us,group3@g.us
4. 选择间隔 (默认值)
5. 选择是否加密 (y/n)
6. 确认执行 (y)
↓
✅ 自动轮番发送
```

### 群发给多个好友

```
1. 选菜单 → 2
2. 输入消息内容
3. 输入好友ID: 8613800000001@c.us,8613800000002@c.us
4. 选择间隔
5. 确认执行
↓
✅ 轮番发送完成
```

### 混合广播 (群+好友)

```
1. 选菜单 → 3
2. 输入消息
3. 输入群ID
4. 输入好友ID
5. 确认执行
↓
✅ 混合发送完成
```

## 📱 ID 获取方法

### 好友 ID

```bash
步骤:
1. WhatsApp中打开与该好友的聊天
2. 按F12打开开发者工具
3. Console执行: window.WAPI.getChatId()
4. 复制结果，格式: 8613800000000@c.us
```

### 群聊 ID

```bash
步骤:
1. WhatsApp中打开群聊
2. 按F12打开开发者工具
3. Console执行: window.WAPI.getChatId()
4. 复制结果，格式: 120363xxx@g.us
```

## 🔄 轮番模式

| 模式 | 说明 | 适用 |
|------|------|------|
| **round-robin** | 依次轮流 | 推荐，均匀分配 |
| **sequential** | 按顺序 | 账号少时 |
| **random** | 随机选择 | 更隐蔽 |

## 💡 常用设置

```javascript
{
    message: '你要发送的消息',
    
    // 群ID列表
    recipients: ['群1@g.us', '群2@g.us'],
    
    // 账号列表 (不填=全部)
    accountIds: ['account1', 'account2'],
    
    // 轮番模式
    rotationMode: 'round-robin',
    
    // 间隔(毫秒)
    intervalBetweenMessages: 1000,
    intervalBetweenRecipients: 1500,
    
    // 加密
    encrypt: true,
    
    // 重试
    retryCount: 3
}
```

## 🐛 快速排查

| 问题 | 解决 |
|------|------|
| 登录失败 | 删除`.wwebjs_auth`，重新扫码 |
| ID格式错 | 检查是否以`@c.us`或`@g.us`结尾 |
| 发送太慢 | 减少`intervalBetweenMessages` |
| 消息被拒 | 增加间隔，减少数量，等待 |
| 加密失败 | 确保`masterSecret`相同 |

## 📞 常见问题

**Q: 多少个接收人最安全？**  
A: 10-20个，间隔标准模式

**Q: 可以发多少个账号？**  
A: 理论无限，推荐3-5个

**Q: 间隔太长怎么办？**  
A: 增加账号数量，分散负载

**Q: 发送后能否编辑？**  
A: 不能，需要重新发送

**Q: 支持群发文件吗？**  
A: 目前仅支持文本，文件另外处理
